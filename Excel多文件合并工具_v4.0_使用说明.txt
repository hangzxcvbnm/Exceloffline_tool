# Excel 多文件合并工具 v4.0 使用说明

> 一键合并多个 Excel 表格，支持纵向追加和横向关联两种模式

---

## 📥 下载与安装

**无需安装！** 下载后直接双击运行：
- 文件：`Excel多文件合并工具_v4.0.exe`
- 大小：约 34 MB
- 系统：Windows 7/10/11

⚠️ **首次运行提示**：Windows 可能显示 SmartScreen 警告，点击 **"更多信息" → "仍要运行"** 即可。这是正常现象（程序未购买代码签名证书）。

---

## 🚀 快速开始

### 1. 启动程序
双击 `Excel多文件合并工具_v4.0.exe`，看到主界面：
- 左侧：文件列表和操作区
- 右侧：运行日志

### 2. 添加文件
点击 **"+ 添加"** 按钮，选择要合并的 Excel 文件（可多选）。

### 3. 选择合并模式

#### 模式一：纵向追加（行堆叠）
适合：多个格式相同的表格，需要汇总到一起

**示例场景**：
- 各班成绩表结构相同（都有 学号、姓名、成绩 列）
- 需要合并成全校总表

**操作**：
1. 选择模式：**纵向追加**
2. 直接点击 **"▶ 开始合并"**

**输出结果**：所有文件的数据按顺序堆叠，自动添加序号列

---

#### 模式二：横向合并（列关联）
适合：不同表格有相同关键列，需要关联在一起

**示例场景**：
- 表 A：学号、姓名、平时成绩
- 表 B：学号、期末成绩
- 需要合并成：学号、姓名、平时成绩、期末成绩

**操作**：
1. 选择模式：**横向合并**
2. 设置**合并键**：输入两表共有的列名（如 `学号` 或 `学号/工号`）
3. （可选）**指定列**：输入需要保留的列关键词，逗号分隔
   - 例如：`学号,姓名,成绩` 只保留包含这些词的列
   - 留空则保留所有列
4. 点击 **"▶ 开始合并"**

**输出结果**：按合并键关联，列并排显示

---

## 📁 文件管理

### 调整文件顺序
- 点击文件右侧的 **↑ ↓** 按钮调整顺序
- 纵向追加时，文件按列表顺序从上到下堆叠

### 删除文件
- 点击文件右侧的 **✕** 按钮移除
- 或点击顶部 **"清空"** 移除所有

---

## 📊 输出文件

合并完成后，自动在桌面生成 Excel 文件：

**文件名格式**：
```
合并结果_纵向追加_20260403_194412.xlsx
合并结果_横向合并_20260403_194415.xlsx
```

> 💡 时间戳确保不覆盖之前的文件

**表格样式**：
- 第 1 行：蓝色标题（合并结果一览表）
- 第 2 行：浅蓝表头（序号、学号/工号、...）
- 第 3 行起：数据行（白蓝交替，便于阅读）
- 首列固定：滚动时序号和学号列始终可见

## ⚙️ 高级技巧

### 列过滤语法（横向合并）
在"指定列"输入框中：
- 多个关键词用**逗号**分隔：`学号,姓名,成绩`
- 支持**模糊匹配**：输入`成绩`可匹配`平时成绩`、`期末成绩`
- 列名有空格也能匹配

### 处理学习通导出的成绩表
本工具针对学习通格式优化：
- 自动跳过第 1 行的"综合成绩"汇总标题
- 从第 2 行的真实表头开始读取
- 自动过滤 "Unnamed" 列

### 大数据量处理
- 建议单次合并不超过 **50 个文件**
- 每个文件不超过 **1 万行**
- 超大文件处理较慢，请耐心等待

## ❓ 常见问题

### Q1：提示"序号列已存在"？
**解决**：程序会自动处理，无需担心。如果源文件已有序号列，会先删除再重新添加，确保序号从 1 开始连续。

### Q2：横向合并没有数据？
**检查**：
1. 合并键是否输入正确（必须与各文件的列名完全一致）
2. 各文件是否真的有相同的列
3. 查看右侧日志，确认匹配了多少行

### Q3：输出文件打不开？
**解决**：
1. 确认电脑已安装 Excel 或 WPS
2. 检查文件是否被其他程序占用
3. 尝试重启程序重新合并

### Q4：纵向追加后数据错位？
**原因**：各文件的列名不完全一致
**解决**：
- 确保所有文件的表头行完全相同
- 使用 Excel 先统一各文件的列名

## 📋 示例工作流程

**场景**：合并 3 个班的成绩表

```
1班成绩.xlsx    2班成绩.xlsx    3班成绩.xlsx
├─ 学号/工号    ├─ 学号/工号    ├─ 学号/工号
├─ 姓名         ├─ 姓名         ├─ 姓名
├─ 平时成绩     ├─ 平时成绩     ├─ 平时成绩
└─ 期末成绩     └─ 期末成绩     └─ 期末成绩
```

**操作步骤**：
1. 点击 **"+ 添加"**，选中 3 个文件
2. 确认文件顺序（1班 → 2班 → 3班）
3. 选择 **"纵向追加"** 模式
4. 点击 **"▶ 开始合并"**
5. 等待 3 秒，桌面生成合并文件

**输出**：包含全部 3 个班数据的汇总表，带序号列
