# Excel 多文件合并工具 v4.0

> 一款轻量级、离线运行的桌面 GUI 工具，支持**纵向追加**和**横向合并**两种模式，轻松将多个 Excel 表格合并为一个。

---

## ✨ 特性

| 功能 | 说明 |
|------|------|
| 🔄 **两种合并模式** | 纵向追加（行堆叠）/ 横向合并（按列关联） |
| 📂 **动态文件管理** | 添加、删除、调整顺序，灵活操作 |
| 🔍 **智能表头定位** | 自动扫描表头行，跳过标题/副标题 |
| 🎛️ **列过滤** | 横向合并时可指定只保留部分列（逗号分隔关键词） |
| 💅 **精美输出排版** | 自动生成带标题行、交替底色、冻结窗格、自动筛选的 Excel |
| 🔌 **离线运行** | 无需网络连接，所有数据处理均在本地完成 |
| 🖥️ **高 DPI 自适应** | 自动检测系统缩放比例，确保界面清晰不模糊 |
| 📦 **EXE 打包即用** | PyInstaller 打包为单文件 EXE，免安装分发 |

## 🚀 快速开始

### 方式一：直接运行 EXE（推荐）

1. 下载 `Excel多文件合并工具_v4.0.exe`
2. 双击运行（首次可能提示 SmartScreen，点击 **更多信息 → 仍要运行**）
3. 添加文件 → 选择模式 → 点击合并

> ⚠️ SmartScreen 提示是因为程序未购买代码签名证书，属于正常现象。

### 方式二：从源码运行

```bash
# 安装依赖
pip install pandas openpyxl

# 运行
python score_merger_gui_v4.py
```

## 📖 使用说明

### 1. 添加文件

| 操作 | 说明 |
|------|------|
| `+ 添加文件` | 选择单个或多个 Excel 文件（支持 `.xlsx` / `.xls`） |
| `📂 浏览文件夹` | 自动导入指定文件夹下的所有 Excel 文件 |
| `清空所有` | 一键清空文件列表 |
| `↑` / `↓` | 调整文件合并顺序 |
| `✕` | 移除指定文件 |

### 2. 选择合并模式

#### 模式一：纵向追加（Stack）

适用场景：**多个结构相同的表格**，需要按行追加。

```
文件 A（甲班）          文件 B（乙班）          合并结果
┌────┬─────┐       ┌────┬─────┐          ┌────┬─────┐
│学号│成绩│       │学号│成绩│          │学号│成绩│
├────┼─────┤       ├────┼─────┤          ├────┼─────┤
│001 │ 90  │       │003 │ 85  │          │001 │ 90  │
│002 │ 80  │       │004 │ 92  │   →      │002 │ 80  │
└────┴─────┘       └────┴─────┘          │003 │ 85  │
                                         │004 │ 92  │
                                         └────┴─────┘
```

#### 模式二：横向合并（Join）

适用场景：**同一批学号的不同维度数据**，需要按列关联。

```
文件 A（班级成绩）      文件 B（期末试卷）        合并结果
┌────┬──────┐       ┌────┬──────┐          ┌────┬──────┬──────┐
│学号│姓名  │       │学号│期末分│          │学号│姓名  │期末分│
├────┼──────┤  +    ├────┼──────┤    →     ├────┼──────┼──────┤
│001 │Alice │       │001 │  90  │          │001 │Alice │  90  │
│002 │Bob   │       │002 │  85  │          │002 │Bob   │  85  │
└────┴──────┘       └────┴──────┘          └────┴──────┴──────┘
```

### 3. 合并选项

| 选项 | 说明 |
|------|------|
| **合并键（列名）** | 横向合并时用来关的列名，默认 `学号/工号`（支持模糊匹配，如 `学号` 即可匹配 `学号` 和 `学号/工号`） |
| **指定列（可选）** | 仅横向合并生效。用逗号分隔需要保留的列名关键词，留空则合并全部列 |

### 4. 执行合并 & 查看结果

确认文件列表和选项无误后，点击 **▶ 一键合并**：
- 合并完成后弹出提示窗口
- 可点击 **📂 打开输出文件夹** 或 **📄 打开结果文件**
- 输出文件自动命名，带时间，不会覆盖已有文件

## 📂 Excel 输出格式

生成的 Excel 表格包含精美的排版：

| 要素 | 标准 |
|------|------|
| 标题行 | "平时成绩情况一览表"，浅蓝底白字，居中对齐 |
| 表头行 | 浅蓝底、黑粗体、边框、居中 |
| 数据行 | 交替底色（白 / 淡蓝），所有列居中排列 |
| 冻结窗格 | 冻结 C3 单元格（标题列冻结） |
| 自动筛选 | 表头行启用筛选 |
| 序号列 | 自动在首列生成 `序号` 列（1 起始） |

## ⚠️ 已知限制

- 仅支持 `.xlsx` 和 `.xls` 格式（不支持 `.csv` / `.ods`）
- 大文件（>50000 行）可能需要较长处理时间
- 首次运行 Windows SmartScreen 提示为正常现象
